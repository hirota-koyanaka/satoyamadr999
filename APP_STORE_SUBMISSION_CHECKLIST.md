## 🔨 ビルドとアップロード

### 1. Xcodeでアーカイブを作成

```bash
# Xcodeでプロジェクトを開く
open login_app/ios/Runner.xcworkspace
```

Xcodeで：
1. Product → Scheme → Runner
2. デバイス選択で「Any iOS Device (arm64)」
3. Product → Archive
4. ビルド完了まで待機

### 2. App Store Connectにアップロード

1. Organizerウィンドウでアーカイブを選択
2. 「Distribute App」をクリック
3. 「App Store Connect」を選択
4. 「Upload」を選択
5. 配布オプションを確認して「Upload」

### 3. ビルドの確認

1. App Store Connectでアプリを選択
2. 「TestFlight」または「App Store」タブを開く
3. 「+」ボタンでビルドを追加
4. アップロードしたビルドが表示されるまで待つ（数分〜1時間）

## 📝 審査提出

すべての情報を入力したら：

1. 「保存」ボタンをクリック
2. すべての必須項目が入力されていることを確認
3. 「審査用に追加」または「審査に提出」をクリック
