# Supabase接続手順（簡易版）

## 手順

### 1. Supabaseダッシュボードでパスワードを確認

1. **Supabaseダッシュボードにアクセス**
   - https://supabase.com/dashboard/project/skbvryjtjxmdslfwivqh/settings/database

2. **Database passwordを確認**
   - 「Database password」セクションを探す
   - パスワードが表示されていない場合は、「Reset database password」をクリックして新しいパスワードを設定
   - **パスワードをメモしておく**

### 2. 接続文字列を作成

以下のコマンドを実行して、対話形式で接続文字列を作成：

```bash
./接続文字列作成.sh
```

または、手動で `.env` ファイルを作成：

```bash
# プロジェクトルートに .env ファイルを作成
cat > .env << EOF
DATABASE_URL=postgresql://postgres:your-password@db.skbvryjtjxmdslfwivqh.supabase.co:5432/postgres
EOF
```

**重要**: `your-password` を実際のパスワードに置き換えてください。

### 3. 接続をテスト

```bash
source venv/bin/activate
python 接続テスト.py
```

### 4. マイグレーションを実行

接続が成功したら：

```bash
python manage.py migrate
```

## 接続文字列の形式

```
postgresql://postgres:[PASSWORD]@db.skbvryjtjxmdslfwivqh.supabase.co:5432/postgres
```

- `[PASSWORD]`: Supabaseダッシュボードで確認したパスワード
- パスワードに特殊文字（`@`、`:`など）が含まれる場合は、URLエンコードが必要です

## トラブルシューティング

- **パスワードが見つからない場合**: Settings → Database → Database password で「Reset database password」をクリック
- **接続できない場合**: パスワードと接続文字列を再確認してください
